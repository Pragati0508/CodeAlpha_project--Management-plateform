# 🚀 ProjectFlow - Smart Project Management Platform

ProjectFlow is a full-stack project management and team collaboration platform that helps teams manage projects, assign tasks, communicate in real-time, and track progress efficiently.

## 📌 Features

### 🔐 Authentication

* User Signup & Login
* JWT Authentication
* Secure Password Hashing using Bcrypt
* Protected Routes

### 📁 Project Management

* Create Projects
* View All Projects
* Manage Project Details

### 📋 Task Management

* Create Tasks
* Assign Tasks to Team Members
* Set Priority Levels
* Track Task Status
* Due Date Management
* Task Search Functionality

### 💬 Real-Time Team Chat

* Socket.IO Powered Chat
* Real-Time Messaging
* Online User Tracking
* User Name Display
* File Sharing Support

### 📊 Analytics Dashboard

* Total Projects
* Total Tasks
* Completed Tasks
* Pending Tasks
* Task Completion Rate
* Interactive Charts

### 👥 Team Members

* User Directory
* Team Collaboration

### ⚙️ Settings

* Update Profile Name
* Change Password
* User Information Management

## ⚡ Installation

### Clone Repository

```bash
git clone https://github.com/Pragati0508/CodeAlpha_project--Management-plateform.git
```

### Backend Setup

```bash
cd Backend
npm install
npm run dev
```

### Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside Backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
